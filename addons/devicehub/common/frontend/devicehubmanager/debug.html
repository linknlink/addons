DHM APP 接口调试
  • 获取 License
  • License 状态查询
  • 激活状态查询
  • 触发激活同步
  • 发现局域网设备 (mDNS)
  • 获取 Entity 列表
  • 添加 Entity
  • 获取当前用户信息
  • 获取系统状态信息
POST /manager/license/get
GET /manager/license/status

查询本地缓存的所有 License 记录。

GET /manager/activation/status

查询当前设备的激活状态以及到期时间。该接口会自动按需触发同步。

POST /manager/activation/sync

手动强制触发云端设备激活状态同步。

GET /manager/device/discover

通过 mDNS 服务发现局域网内 iSG 等所有 _linknlink._tcp 形式的设备，过程可能需要 3 秒钟。

GET /manager/device/entities

获取 HA 中的所有 Entity 分组信息 (代理至 ha2devicehub)

POST /manager/device/entities

向局域网广播添加 Entity 请求 (代理至 ha2devicehub)

GET /manager/account/info
GET /manager/system/info
