HA 访问令牌

在 Home Assistant → 用户设置 → 安全 中创建长效访问令牌

如何获取 HA Token？

  1. 打开 Home Assistant 界面
  2. 点击左下角头像 → 用户设置
  3. 滚动到底部 → 安全 → 长效访问令牌
  4. 点击 创建令牌，输入名称后复制令牌
