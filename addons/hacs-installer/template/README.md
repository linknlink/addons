# HACS Installer

HACS Installer 是一个 Haddons Addon，旨在为 Home Assistant 提供便捷的 HACS 安装和管理能力。

## 概述

HACS Installer 为 Home Assistant 用户提供了一键安装、管理和卸载 HACS (Home Assistant Community Store) 的功能。通过简洁的 Web 界面，用户可以轻松完成 HACS 的部署，无需手动执行命令行操作。

## 核心能力

- **一键安装**：自动化下载并安装最新版本的 HACS
- **智能管理**：自动检测安装状态，支持重新安装
- **卸载支持**：提供一键卸载功能，清理残留文件

## 主要功能

- ✅ 自动安装：从 GitHub 官方源获取最新 HACS 并安装到指定目录
- ✅ 状态检测：实时检查 HACS 是否已安装
- ✅ 一键卸载：安全移除 HACS 及其相关文件
- ✅ 网络优化：内置网络连接检查，确保安装过程顺畅
- ✅ 端口配置：使用 8202 端口提供服务

## 适用场景

- 需要快速在 Home Assistant 中启用社区商店功能
- 不熟悉命令行操作，希望通过图形界面管理 HACS
- 需要在多台设备上快速部署 HACS
- 需要方便地卸载和重装 HACS 进行故障排查
