# HACS Installer

HACS Installer is a Haddons Addon designed to provide convenient HACS installation and management capabilities for Home Assistant.

## Overview

HACS Installer provides Home Assistant users with one-click installation, management, and uninstallation of HACS (Home Assistant Community Store). Through a simple Web interface, users can easily complete HACS deployment without manually executing command-line operations.

## Core Capabilities

- **One-Click Installation**: Automatically download and install the latest version of HACS
- **Smart Management**: Automatically detect installation status and support reinstallation
- **Uninstallation Support**: Provide one-click uninstallation to clean up residual files

## Main Features

- ✅ Automatic Installation: Get the latest HACS from the official GitHub source and install it to the specified directory
- ✅ Status Detection: Real-time check if HACS is installed
- ✅ One-Click Uninstall: Safely remove HACS and its related files
- ✅ Network Optimization: Built-in network connection check to ensure a smooth installation process
- ✅ Port Configuration: Uses port 8202 to provide service

## Use Cases

- Need to quickly enable community store functions in Home Assistant
- Unfamiliar with command-line operations, prefer to manage HACS through a graphical interface
- Need to quickly deploy HACS on multiple devices
- Need to conveniently uninstall and reinstall HACS for troubleshooting
