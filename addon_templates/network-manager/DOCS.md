# Network Manager 使用说明

本文档详细介绍如何在生产环境中使用 Network Manager Addon。此文档会显示在 Haddons Web 界面的"文档"标签页中，面向最终用户（ToC 产品用户）。

## 快速开始

### 安装和启动

1. 在 Haddons Web 界面中找到 Network Manager
2. 点击"安装"按钮安装 Addon
3. 安装完成后，在"配置"标签页中配置必要的选项（如需要）
4. 点击"保存"保存配置
5. 点击"启动"按钮启动 Addon

### 首次配置

首次使用时，建议在"配置"标签页中设置以下选项：

- **配置项 1**：说明该配置项的作用和推荐值
- **配置项 2**：说明该配置项的作用和推荐值
- **配置项 3**：说明该配置项的作用和推荐值

## 配置说明

### 配置选项说明

在 Haddons Web 界面的"配置"标签页中，您可以配置以下选项：

| 配置项 | 类型 | 说明 | 默认值 | 必填 |
|--------|------|------|--------|------|
| `option1` | string | 配置项 1 的说明 | `default1` | 否 |
| `option2` | number | 配置项 2 的说明 | `10` | 否 |
| `option3` | boolean | 配置项 3 的说明 | `true` | 否 |

### 配置示例

**示例 1：基本配置**

```json
{
  "option1": "value1",
  "option2": 20,
  "option3": true
}
```

**示例 2：高级配置**

```json
{
  "option1": "value1",
  "option2": 30,
  "option3": false
}
```

## 注意事项

### 使用前检查

- ✅ 确保系统满足 Addon 的运行要求
- ✅ 检查必要的资源（端口、存储空间等）是否可用
- ✅ 确认网络连接正常（如需要）

## 故障排查

### 常见问题

#### 问题 1：Addon 无法启动

**可能原因**：
- 配置错误
- 资源不足
- 端口冲突

**解决方法**：
1. 检查"配置"标签页中的配置是否正确
2. 查看"日志"标签页中的错误信息
3. 确认系统资源是否充足

#### 问题 2：功能异常

**可能原因**：
- 配置不正确
- 网络问题
- 依赖服务未启动

**解决方法**：
1. 检查相关配置项
2. 查看日志获取详细错误信息
3. 确认依赖服务状态

### 获取帮助

如果遇到问题无法解决，可以：

1. 查看"日志"标签页获取详细错误信息
2. 检查配置是否正确
3. 联系技术支持

## 更新和维护

### 更新 Addon

当有新版本可用时，Haddons 会在"信息"标签页中提示更新。点击"更新"按钮即可升级到最新版本。

### 备份配置

建议定期备份"配置"标签页中的配置信息，以便在需要时快速恢复。
